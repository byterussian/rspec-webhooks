# Rspec::Webhooks

rspec-webhooks will provide useful helpers for webhooks testing. WIP, stay tuned.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'rspec-webhooks'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install rspec-webhooks
